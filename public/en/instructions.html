The Prisoner's Dilemma Instructions

This game is played in rounds by two human players randomly paired.

In each round, both players must make the choice to cooperate or defect. If both players choose to collaborate, they each receive 3 coins. If both players defect, they each receive 1 coins. If one choose to cooperate and the other doesn't, the cooperator receives 0 coins while the defector receives 5 coins. The game is repeated 5 rounds.

If you understood the instructions correctly press the DONE Button to proceed to the game.
